# Website Screenshot Generator

A simple **website screenshot generator** built with **Python** and **Flask**. This app allows users to enter any website URL and instantly get a screenshot of that page, powered by a screenshot API.

---

## Features

- Simple web interface using Flask
- Accepts user input through an HTML form
- Integrates with an external API to capture website screenshots
- Dynamically displays screenshots on the web page
- Lightweight and beginner-friendly

---

## Built With

- [Python](https://www.python.org/)
- [Flask](https://flask.palletsprojects.com/)
- Screenshot API of your choice (e.g., [Screenshotbase](https://screenshotbase.com/))
